# Week 09 Mini Project Brief

## Title
Time-series baseline forecasting notebook

## Problem
Build and compare baseline time-series models with proper diagnostics.

## Data
- Synthetic AR data
- Market index returns
- Volatility proxy

## Steps
1. Prepare and transform series
2. Fit baseline models
3. Run residual diagnostics
4. Evaluate forecasts
5. Document practical takeaway

## Metrics
- MAE
- RMSE
- Residual whiteness
- Economic usefulness

## Common Mistakes to Avoid
- Hidden leakage from future data
- Ignoring transaction and execution effects
- Reporting only best-case outcomes
