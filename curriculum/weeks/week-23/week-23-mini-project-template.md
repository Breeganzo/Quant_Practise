# Week 23 Mini Project Brief

## Title
Strategy pitch deck

## Problem
Build and present a concise strategy pitch with risk-first framing.

## Data
- Strategy metrics
- Risk scenarios
- Implementation assumptions

## Steps
1. Define thesis and edge
2. Quantify expected behavior
3. Add risk controls and invalidation
4. Prepare 10-minute deck
5. Run mock Q&A

## Metrics
- Pitch clarity
- Risk defense quality
- Q&A resilience
- Narrative consistency

## Common Mistakes to Avoid
- Hidden leakage from future data
- Ignoring transaction and execution effects
- Reporting only best-case outcomes
